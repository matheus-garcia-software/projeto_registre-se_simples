# 🚀 Projeto Registro Simples com Validação Dinâmica

Este é um projeto de formulário de cadastro moderno, desenvolvido para praticar manipulação de DOM, eventos de formulário e integração de bibliotecas externas em JavaScript.

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estruturação semântica.
- **CSS3**: Estilização com temática Neon e Design Responsivo.
- **JavaScript**: Lógica de validação e controle de eventos.
- **IMask.js**: Biblioteca para máscaras dinâmicas de CPF e Celular.

## 🌟 Funcionalidades
- [x] Validação de campos obrigatórios.
- [x] Máscara inteligente para CPF (`000.000.000-00`).
- [x] Máscara inteligente para Celular (`(00) 00000-0000`).
- [x] Feedback visual de erro e sucesso para o usuário.
- [x] Interface totalmente responsiva.
